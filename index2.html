Folder ./module/block-in-viewport

не забудь подключить jquery-migrate-3.0.0.js так как ошибка с новой версией jquery возникает

Доступны следующие опции:

  • activeClass – класс, который добавляется, когда блок появляется по вьюпорте (по умолчанию – “in-viewport”)
  • once – флаг, который определяет добавлять активный класс единожды или при каждом появлении блока во вьюпорте (по умолчанию – true)
  • holder – родительский блок, который скроллит виджеты. Можно не задавать – тогда скрипт привяжет виджет к скроллу окна (по умолчанию – окно)
  • visibleMode – опция, которая позволяет точно настроить момент, в который активный класс будет добавлен. Существует 4 настройки (По умолчанию – 1)
    • 1 – класс добавляется, когда блок полностью показывается по вьюпорте
    • 2 – когда хотя бы половина блока во вьюпорте
    • 3 – хотя бы пиксель блока во вьюпорте
    • NUMBER – любое другое число. Кастомная настройка в пикселях. Если поставить 20 – то как только блок показывается на 20 пикселей на экране – активный класс добавится
  • Независимо от опции visibleMode активный класс будет добавлен, если блок своим низом или верхом достиг середины окна

Folder ./module/scroll-text

Данный плагин позволяет реализовать циклическое прокручивание текста в пределах блока. Верстка под него очень похожа на верстку под карусель. Тоже есть маска и широкий блок, который содержит проскролливаемый текст (один или несколько абзацев).

HTML-структура:

	<div class="line-box"> <!-- Родительский блок, к которому будет подключаться плагин (может быть одновременно и маской) -->
		<div class="line-wrap"> <!-- Блок, который будет скроллится -->
					<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p> <!-- Контент бегущей строки -->
		</div>
	</div>

В плагине доступны следующие опции:

	jQuery('div.line-box').marquee({
		mask: null,               // селектор маски (опционально, по умолчанию маской будет блок, к которому применён плагин)
		line: 'div.line-wrap',    // селектор скроллящегося блока (по умолчанию прямой потомок маски)
		items: '>*',              // селектор элементов в скроллящемся блоке (по умолчанию прямой потомок скролл.блока)
		animSpeed: 50,            // скорость анимации (пикселей в секунду)
		pauseOnHover: true,       // приостанавливать бегущую строку на ховер
		hoverClass: 'hover',      // класс ховерного состояния бегущей строки
		direction: 'left',        // направление движения "влево" или "вправо" ("left","right")
		initialDelay: 0           // первоначальная задержка перед стартом бегущей строки (мсек)
	});
